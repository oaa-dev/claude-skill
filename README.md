# Personal Plugins

A personal Claude Code plugin marketplace.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| knowledge-garden | Project knowledge management for Laravel and Next.js. 1 agent, 3 commands, 5 skills. | `/plugin install knowledge-garden` |

## Claude Code Install

```bash
/plugin marketplace add https://github.com/betrnk/personal-plugins
/plugin install knowledge-garden
```

## Cursor Install

```text
/add-plugin knowledge-garden
```
